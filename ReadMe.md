#  Defeat Evil Wizard Game

##  Overview
This project is a beginner-friendly turn-based battle game built with Python using Object-Oriented Programming (OOP). Players can choose from multiple hero classes and battle against the powerful Evil Wizard using attacks, healing abilities, and special powers.

The game demonstrates important programming concepts such as classes, inheritance, methods, random damage generation, and user interaction through menus.

---

#  Features

- Four playable character classes:
  - Warrior
  - Mage
  - Archer
  - Paladin

- Unique abilities for each class

- Turn-based combat system

- Healing mechanics

- Randomized attack damage

- Evil Wizard enemy with regeneration abilities

- Victory and defeat messages

---

#  Technologies Used

- Python 1
- Object-Oriented Programming (OOP)
- Random module

---

#  How to Play

1. Run the Python script 
```bash
python game.py
```
2. Choose your character class.
3. Select actions during battle:
   - Attack
   - Heal
   - Use Ability 1
   - Use Ability 2
   - View Stats
4. Defeat the Evil Wizard before your health reaches zero.

---

#  Character Classes

## Warrior
- Power Strike → Heavy damage attack
- Battle Cry → Increases attack power

## Mage
- Fireball → Powerful magic attack
- Lightning Bolt → Fast electric attack

## Archer
- Quick Shot → Double arrow attack
- Evade → Avoids the next enemy attack

## Paladin
- Holy Strike → Bonus holy damage
- Divine Shield → Blocks the next attack

---

#  Evil Wizard Abilities

- Regenerates health after each turn
- Uses Dark Magic attacks against the player
